# Getting Started
## Tech Stack
- Django
- React
- PostgreSQL
- TailwindCSS

## Backend Setup
```
$ cd PetPal/backend
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ python manage.py makemigrations
$ python manage.py migrate
$ python manage.py runserver
```

## UI Setup
```
$ cd PetPal/frontend
$ npm install
$ npm start
```
